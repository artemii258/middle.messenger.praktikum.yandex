## Запуска проекта

+ `npm run start`		запуск проекта из папки dist с помощью express
+	`npm run dev`     запуск проекта в режиме разработки
+	`npm run clean`  очистка папки dist
+ `npm run build`		сборка проекта в режиме продакшена
